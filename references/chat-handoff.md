# ChatGPT Handoff

When local calculation is unnecessary, return:

1. `Why hand off`: one sentence explaining why the runtime adds little value.
2. `Prompt`: a self-contained prompt ready to paste into ChatGPT.
3. `Snapshot`: only facts required to answer it.
4. `Boundary`: claims that still require local PoB or in-game verification.

Do not attach the full build XML, raw result JSON, or complete roadmap. Prefer
current level, class, relevant mechanics, the decision being considered, and
already measured values. Keep the snapshot short enough to inspect manually.
